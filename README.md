# 🎀 Invitación Cumpleaños de Dayana 👑

## 📁 Estructura del proyecto

```
mi-proyecto/
├── index.html          ← página principal
├── assets/
│   ├── music/
│   │   └── cancion.mp3 ← pon aquí tu archivo de música
│   └── img/
│       └── dayana.jpg    ← (opcional) foto de Anna
└── README.md
```

## 🎵 Cómo agregar tu música

1. Copia tu archivo `.mp3` dentro de `assets/music/`
2. En `index.html` busca la línea:
   ```html
   <source src="assets/music/cancion.mp3" type="audio/mpeg"/>
   ```
3. Cambia `cancion.mp3` por el nombre de tu archivo

## 🚀 Cómo subir a GitHub + Vercel

1. Crea un repositorio en GitHub
2. Sube TODA la carpeta (index.html + assets/)
3. En vercel.com → "Add New Project" → conecta el repo → Deploy
4. ¡Listo! Comparte el link con todos 🎉
